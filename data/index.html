If analyzing data is half the battle, and getting it is the other half, then managing it once you have it is the third half. These two episodes look at a few rules for doing that, and at one tool that can be used to help.

{% include toc.html %}
  • Store data in a hierarchy of folders and files with regular names that can easily be pattern-matched.
  • Use README files to store metadata.
  • Have the computer track the programs used to process data.
